# Student Management System

A simple Student Management System developed using Python and Object-Oriented Programming (OOP). This project helps manage student records with a clean and organized code structure.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Student Photo Upload
- Clean OOP Structure

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Visual Studio Code

## How to Run

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Run the Python file.

```bash
python student_management_system.py
```
